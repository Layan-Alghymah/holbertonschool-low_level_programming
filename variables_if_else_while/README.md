# C - Variables, if, else, while

This project covers the fundamentals of the C programming language, including:

- Variables
- Data types
- Arithmetic operators
- Relational operators
- Logical operators
- if, else, and else if statements
- while loops
- ASCII characters

## Requirements

- Ubuntu 20.04 LTS
- GCC
- Compilation flags:
  ```bash
  gcc -Wall -Werror -Wextra -pedantic -std=gnu89
  ```
